  1. Front End Application setup process.
    2. Open command window in the root directory of this project & Go to front Folder then Type npm install for windows or yarn for mac and hit enter. To run this project locally run " npm run start in the command window. Project will be hosted on localhost:3000.

    3. Open a another command window in the root directory of this project & Go to back Folder then Type npm install for windows or yarn for mac and hit enter. To run this project locally run " npm run start in the command window. Application API will be hosted on localhost:5000.
